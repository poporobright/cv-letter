HTML and CSS Resume Project

A clean and structured recreation of a professional resume layout using HTML and CSS. This project focuses on using modern layout practices, avoiding outdated elements, and keeping the code neat and organized.

Preview

The layout of this project is built to match the structure and specifications of the file named image_3552b4.jpg. It features clean text sizes, proper spacing, and uses a central box container to make sure all the text naturally lines up perfectly on the left side of the page, exactly like a physical sheet of paper.

Features

    Standard HTML5 tags are used to build a solid structure.

    A central container box handles the spacing automatically, so you do not have to guess and manually change the left margins for every single line of text.

    The structure is clean and easy to read, making it simple to update your contact details or add new jobs later.

Technology Used

    HTML5

    CSS3

Project Structure

    README.txt (This instructions file)

    index.html (The file containing your resume code and styling)

    image_3552b4.jpg (The original design image being copied)

How to Run This Locally

    Download or clone this folder to your computer.

    Open the folder and locate the index.html file.

    Double-click the index.html file, and it will automatically open and display your resume in your default web browser.
